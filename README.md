# Orbital Forge Website

Static GitHub Pages site for Orbital Forge, LLC.

## Public positioning

The primary website presents Orbital Forge as a Colorado small business conducting independently conceived and privately funded integrated photonics and RF-photonics research and development.

The public site intentionally limits technical disclosure. Detailed architecture, performance objectives, diagrams, patent materials, customer information, submissions, and roadmaps should not be published without appropriate review.

## Main routes

- `/` — corporate photonics and RF-photonics landing page
- `/network/` — legacy blockchain and validator operations
- `/network/wallet-tracker.html` — wallet tracker

Legacy top-level pages redirect to the current public routes.

## Publishing

Copy the contents of this folder into the root of the GitHub Pages repository, preserving the repository's `.git` directory, then commit and push.
